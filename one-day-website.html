    Website jargon.

    Ugh, we hate using tech-talk and website jargon, but sometimes it's unavoidable. We're not trying to confuse you, so here are some explanations you might find helpful.

    • Website pages

      When talking about the size of a website, we refer to it by the number of "pages" it contains. Your One Day Website includes five pages. As an example, these might be:

      1. Home
      2. About
      3. Services
      4. Blog
      5. Contact

      The exact pages you want are up to you. If you'd like some help, we can advise you on the pages we think would be best.

    • Sitemap

      Following on from the last point, a collection of pages is called a "sitemap". Using the same example, your complete sitemap for your website might be:

      1. Home
      2. About
      3. Services
      4. Blog
      5. Contact

      Your sitemap is totally up to you. Structuring your website and sitemap falls into the expertise of our Copywriter, so we can assist here if needed. 

    • Content Management System

      A "Content Management System" — also known as CMS — its the software used to build and edit the website. For example, all our One Day Websites are built in Duda. However, larger websites can be built by us in WordPress. Both are built without code, which means it's easy for you to use and edit after the build is complete.

    • Duda

      Duda is the Content Management System (CMS) we build One Day Websites in. We have purposely chosen this CMS as it's exceptionally easy to use, edit, and looks great. We love it so much that our own Helium website — the website you're on right now — is built in Duda!


      Duda is a code-free CMS, which means you'll never need to hire technical experts or Developers to help you. You can have full control of the website once the build is complete and won't need to rely on us for future updates. Of course, we're always available to help you if you need us!

    • SEO

      SEO stands for Search Engine Optimisation. It's the process of getting your website picked up by search engines, like Google, with the goal of appearing on the first page of search engine results. SEO is an extremely time intensive (but worth it) marketing strategy that takes months of work before you start seeing results. 


      However, we implement basic SEO during your One Day Website build to help you get a head start. We can also provide ongoing SEO to help you stand out on Google, if that's a priority for you.
